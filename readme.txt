== Description ==

Available for the all Wordpress platform. The Simple Linkedin Feeds is available free of charge and can be downloaded and installed in minutes of time without obligation. Those who choose to use the Simple Linkedin Feeds will find themselves met with an array of awesome features.

Some of the great features and benefits that will come your way when you download the Simple Linkedin Feeds to your website include:

(i) Show/Hide Frontend view using setting.

(ii) Use Shortcode

Easy to use Different pages or posts or widget using shortcode "[sldw_simple_linkedin_widget]".

Use This Code in Page Template <code><?php echo do_shortcode('[sldw_simple_linkedin_widget]'); ?></code>.

The Simple Linkedin Feeds can easily be added to your site with all of the above benefits.



== Installation ==


Automatic installation through WordPress:


- Log-in to your WordPress site.

- Move mouse over Plugins and click Add New.

- Under Search type in Simple Linkedin Feeds, then click Search Plugins.

- At the results page, click Install Now.

- Once installed, click Activate Plugin. You're done!



Manual installation:



- Upload the /wp-content/plugins/directory.

- Go to your WordPress dashboard and navigate to Plugins -> Installed Plugins.

- At the Plugin page in your WordPress Administration area, activate the Simple Linkedin Feeds plugin.
